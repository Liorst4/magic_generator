#+TITLE: Magic Generator

Generate unique hexadecimal numbers based on leet speak.

* Usage
#+BEGIN_SRC bash
  ./magic_generator.sh -l $NUMBER_OF_DIGITS

  # 16 bit number
  ./magic_generator.sh -l 4

  # 32 bit number
  ./magic_generator.sh -l 8

  # 64 bit number
  ./magic_generator.sh -l 16
#+END_SRC
